# Built 4 More

Premium marketing site for **Built 4 More** — a youth leadership and athlete development program for middle school and high school students.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com) v4
- [Framer Motion](https://www.framer.com/motion/)

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero, mission, pillars, mindset, testimonials, parent trust, CTA |
| `/mission` | Full mission, values, and pillars |
| `/programs` | Foundation, Elite Development, Summit Sessions |
| `/about` | About Abir — founder story |
| `/faq` | Parent FAQ accordion |
| `/contact` | Contact form and info |

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Customization

- **Content & copy:** `src/lib/constants.ts`
- **Colors & theme:** `src/app/globals.css`
- **Site metadata:** `src/app/layout.tsx`
